# elm-rollup

Elm + Rollup starter kit

## Installation

```console
$ git clone https://github.com/aminnairi/elm-rollup.git my-elm-project
$ cd my-elm-project
```

## Dependencies Installation

```console
$ make install
```

## Development

```console
$ make dev
```

## Production

```console
$ make prod
```

## Shell

```console
$ make shell
```

## Elm Format

```console
$ make format
```
